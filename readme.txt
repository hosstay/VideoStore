For console:
1) inside VideoStore/videoStore/ run "javac *.java"
2) inside VideoStore/ run "java videoStore.RunPrompt"

For IDE:
Make a new project and dump the files in and run it.

info:
RunPrompt is the prompt class that allows you to utilize the StatementSystem.

There are some helper options in the menu for easily populating the database with 5 movies and 2 customers. As well as some options for listing the movie names and types and customer names and rented movies if you forget.

Sample of use:
1) add movies and customers either manually or through helper options.
2) have the customers check out some movies
3) have time pass
4) have the customers check in some movies
5) print the charge statement for the customers.
